# Python学生管理系统 python872

## 项目概述

本项目是基于Django框架的Python学生管理系统，旨在为学校提供一个高效、易用的学生信息管理系统。

## 技术栈

- Python版本：3.11
- 开发框架：Django
- 数据库：MySQL
- 开发工具：PyCharm

## 功能模块

系统主要包括以下三大模块：

### 管理员

- 首页
- 个人中心
- 教师管理
- 学生管理
- 公告信息管理
- 课程类型管理
- 课程信息管理
- 选课信息管理
- 成绩信息管理

### 教师

- 首页
- 个人中心
- 公告信息管理
- 课程信息管理
- 选课信息管理
- 成绩信息管理

### 学生

- 首页
- 个人中心
- 公告信息管理
- 课程信息管理
- 选课信息管理
- 成绩信息管理

## 系统角色

- 管理员：负责系统管理、教师管理、学生管理、课程管理等功能操作。
- 教师：负责公告管理、课程管理、选课管理、成绩管理等功能操作。
- 学生：负责查看公告、课程信息、选课、查看成绩等功能。

## 运行环境

- 操作系统：支持Linux、Windows等
- Python环境：Python 3.11及以上
- 数据库：MySQL

## 部署步骤

1. 安装Python环境
2. 安装Django框架
3. 安装MySQL数据库
4. 导入数据库脚本
5. 运行项目

## 目录结构

```
project/
│
├── apps/                # 应用目录
│   ├── admin/           # 管理员模块
│   ├── teacher/         # 教师模块
│   └── student/         # 学生模块
│
├── media/               # 媒体文件目录
├── static/              # 静态文件目录
├── templates/           # 模板文件目录
│
├── manage.py            # 管理脚本
├── requirements.txt     # 依赖文件
└── settings.py          # 项目配置文件
```
